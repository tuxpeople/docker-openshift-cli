## Image tags
- ```latest``` always refers to the latest tagged release
- There are tags for major, minor and dotreleases (eg. ```1.0.0```, ```1.0```, ```1``` )

## Usage

```sh
docker pull tdeutsch/openshift-cli:<tag>
```

or

```sh
docker pull quay.io/tdeutsch/openshift-cli:<tag>
```
